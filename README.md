## How to use
1. Open [this](https://raw.githubusercontent.com/kaibadash/chatwork_to_all_short/master/chatwork_to_all_short.js).
2. Select all(ctrl+a)
3. Drag and drop selected text to bookmark bar(Add as a bookmarklet)
4. Visit chatwork room
5. Fire!:fire:
